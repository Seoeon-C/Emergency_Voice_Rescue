LLM 응급처치 응답 컴포넌트 MVP 정의서 및 체크리스트

MVP 목표

산악 응급 상황에서 환자 증상을 입력받아 응급처치 방법을 안내하는 LLM 응답 기능 구현

체크리스트

  [x] 헬스케어 QA 데이터 수령
  [x] 응급/산악 관련 카테고리 필터링 및 CSV 변환 (build_emergency_train.py, build_emergency_val.py)
  [x] 데이터 전처리 (cleaned_data.py)
  [x] FirstAid 데이터와 병합 (merge_datasets.py)
  [x] Llama 3.2-1B QLoRA 샘플 학습 실험 (finetune_llama32_sample.ipynb)
  [x] 샘플 추론 결과 확인
  [ ] 전체 데이터 파인튜닝 완료
  [ ] 최종 추론 품질 검증

MVP 성공 기준

  - 발목 염좌, 저체온증, 호흡 곤란 질문에 응급처치 안내가 출력된다.
  - 동일 문장이 반복되지 않는다.
  - 질문과 관련 없는 내용이 출력되지 않는다.
